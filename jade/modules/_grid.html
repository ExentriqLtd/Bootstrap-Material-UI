Responsive Grid

Columns, gutters, and margins

The Exentriq Bootstrap Material UI layout grid is made up of three elements: columns, gutters, and margins.

Column width is defined using percentages(Bootstrap), rather than fixed values, to allow content to flexibly adapt to any screen size. The number of columns displayed in the grid is 12.

Gutter widths are fixed values at each breakpoint range.

Margins are the space between content and the left and right edges of the screen.Margin widths are defined as fixed values at each breakpoint range.

For more information visit Bootstrap (v3.3.5) Website.
