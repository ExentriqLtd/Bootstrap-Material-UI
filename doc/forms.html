Text Area

Add the class eq-ui-textarea in the textarea.


<form>
    <div class="row">
        <div class="col-md-12">
            <div class="form-group eq-ui-input-field">
                <textarea id="textarea_field" rows='1' class="form-control eq-ui-textarea"></textarea>
                <label for="textarea_field">Textarea</label>
            </div>
        </div>
    </div>
</form>

Validate Fields

You can add custom validation messages by adding either data-error or data-success attributes to your input field labels.


<form>
    <div class="row">
        <div class="col-md-12">
            <div class="form-group eq-ui-input-field">
                <input id="email_val" type="email" class="form-control validate">
                <label for="email_val" data-error="wrong" data-success="right">Email</label>
            </div>
        </div>
    </div>
</form>
